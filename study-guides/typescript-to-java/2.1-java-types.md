# 2.1 Java Types

## Primitive vs Reference Types
- Primitive: int, boolean, etc.
- Reference: Objects, Arrays, etc.

## Arrays and Type Casting
```java
// Arrays
int[] numbers = {1, 2, 3};

// Type casting example
double d = 3.14;
int i = (int) d; // casts double to int
```

## Generics in Java
```java
// Generic List
List<String> names = new ArrayList<>();
```
