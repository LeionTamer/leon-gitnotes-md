# Color theme

Color Name	Original (Light)	Dark Mode Hex	Usage in Dark Mode
Background	#EBE3D0 (Paper)	#1A1818	Deep Charcoal. Avoid pure black (#000); this has a hint of warmth.
Text Color	#2C2B2B (Ink)	#EBE3D0	Paper White. We essentially flip the background color to be the text color.
Primary Accent	#CD3A33 (Red)	#A32822	Velvet Red. Slightly darker/richer so it doesn't vibrate too hard against the dark background.
Highlights	#F4C430 (Gold)	#D4A017	Metallic Gold. Used for borders and headlines to give a "glowing" effect.
Secondary	#6B9FAB (Teal)	#2F4F5D	Midnight Teal. Used for subtle backgrounds or footer sections.
